사용법
파일명 뒤에 #"분류명"을 적으면 "분류명"폴더가 생성되며 그 안에 #"분류명"을 적은 파일이 들어가게 된다

    *주의*
    반드시 파일명 가장뒤에 #"분류명"을 적어야 정상적인 작동이 가능하다

EX

> 파일명
>
> > 안녕하세요 #2020
> > 어서오세요 #2020
> > test #2021
> > 2019년 사진 #사진
> > 2018년 사진 #사진
> > 4581291.jpg #사진

---> run main.py --->
